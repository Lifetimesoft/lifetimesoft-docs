# การเพิ่มบัญชี AI (Add Account)

วิธีการเพิ่มบัญชี AI เข้าสู่ระบบ LifeTimeSoft เพื่อให้ AI Agents สามารถเรียกใช้งานโมเดล AI จากผู้ให้บริการภายนอกได้

## ขั้นตอนการเพิ่มบัญชี

### 1. กดปุ่ม "เพิ่มบัญชี" (Add)
ไปที่หน้าการจัดการบัญชี AI (AI Account Management) จากนั้นคลิกที่ปุ่ม **เพิ่ม (Add)**

### 2. เลือกผู้ให้บริการ (Provider)
เลือกผู้ให้บริการ AI ที่คุณต้องการเชื่อมต่อ (เช่น OpenAI, Gemini, Anthropic เป็นต้น)

### 3. กรอก API Key
นำ API Key ที่ได้จากหน้า Developer Dashboard ของผู้ให้บริการมาใส่ในช่องที่กำหนด โปรดตรวจสอบให้แน่ใจว่า API Key นั้นมีสิทธิ์การใช้งานที่เพียงพอ

### 4. บันทึกและตรวจสอบ
คลิกปุ่ม **บันทึก (Save)** ระบบจะทำการตรวจสอบความถูกต้องของ API Key หากสำเร็จ บัญชีจะไปปรากฏในหน้ารายการบัญชี และพร้อมให้ AI Agents ใช้งานได้ทันที
