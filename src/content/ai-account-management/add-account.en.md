# Add AI Account

How to add a new AI Account to your LifeTimeSoft system to allow AI Agents to utilize external AI models.

## Steps to Add an Account

### 1. Click "Add Account"
Navigate to the AI Account Management page and click on the **Add** button.

### 2. Select Provider
Choose the AI Provider you want to connect (e.g., OpenAI, Gemini, Anthropic).

### 3. Enter API Key
Input the API Key obtained from the provider's developer dashboard. Ensure the key has the necessary permissions.

### 4. Save and Verify
Click **Save**. The system will verify the API key and, if successful, the account will appear in your account list and be ready for use by your AI Agents.
