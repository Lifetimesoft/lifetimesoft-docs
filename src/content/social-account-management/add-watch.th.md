# เพิ่ม Watch Account

วิธีการเพิ่ม Watch Account เพื่อทำการดูและติดตาม Social Account ต่าง ๆ ที่ต้องการ

## ขั้นตอน

### 1. กดปุ่ม เพิ่ม ในส่วนของ Watch Account

![watch-add-btn](https://static.lifetimesoft.com/docs/img/watch-add-btn.png)

### 2. เลือก Platform แล้วใส่ ID จากนั้นกด Fetch Data

![add-social-account](https://static.lifetimesoft.com/docs/img/add-social-account.png)

### 3. ระบบจะ Fetch ข้อมูลและแสดงตัวอย่าง เมื่อยืนยันแล้วกด Import Channel Data

![add-social-account-import](https://static.lifetimesoft.com/docs/img/add-social-account-import.png)

### 4. ระบบจะแสดง list account ทั้งหมดที่ import เข้ามา

![watch-account-list](https://static.lifetimesoft.com/docs/img/watch-account-list.png)
